Process of Hacking

    There are mainly 5 process in Hacking :


    1. Reconnaissance :

      2. Reconnaissance is also called as Footprinting and Information gathering process/phase. In this process the attacker gathers information about the target.

        There are two types of Reconnaissance :

          a. Active Reconnaissance : It involves direct communication with the target to get information. Eg : Using Nmap tool to scan the target.

          b. Passive Reconnaissance : It involves indirect methods of collecting information about the target. Eg : Collecting information from Social media, onion sites, public websites etc.


    2. Scanning :

      3. Scanning is a set of procedures for identifying live hosts, ports, and services, discovering Operating system and architecture of target system, Identifying vulnerabilities and threats in the network. Some of the tools used in this process are Nessus, Nexpose and NMAP.

        Scanning has three types :

          a. Port scanning – used to list open ports and services.

          b. Network scanning – used to list IP addresses.

          c. Vulnerability scanning – used to discover the presence of known vulnerabilities.


    3. Gaining Access :

      4. The actual hacking takes place at this process where hack gets successful in gaining access to the target system, network or application using the potential tools and techniques.The main tool used in this process is Metasploit.


    4. Maintaining Access :

      5. In this process, the access to a system is gained already by the hacker. Some backdoors are installed such that the hacker can access the system when he desires access in owned system in future. Metasploit is the preferred tool in this process.


    5. Clearing Tracks :

      6. Clearing Tracks is an unethical activity. It has to do with the deletion of logs of all the activities that take place during the hacking process.
