    1. Acunetix

      2. Acunetix is an automated web application security testing and ethical hacking tool. It is used to audit your web applications by checking for vulnerabilities like SQL Injection, cross–site scripting, and other exploitable vulnerabilities. In general, Acunetix scans any website or web application that is accessible via a web browser and uses the HTTP/HTTPS protocol.



      Acunetix offers a strong and unique solution for analyzing off–the–shelf and custom web applications including those utilizing JavaScript, AJAX and Web 2.0 web applications. Acunetix has an advanced crawler that can find almost any file. This is important since what is not found cannot be checked.

    5. Nikto

      6. Nikto is another favorite, well–known as part of the Kali Linux Distribution. Other popular Linux distributions such as Fedora already come with Nikto available in their software repositories as well. This security tool is used to scan web servers and perform different types of tests against the specified remote host. Its clean and simple command line interface makes it really easy to launch any vulnerability testing against your target.

       Nikto's main features include :

      1. Detects default installation files on any operating system

      2. Detects outdated software applications

      3. Integration with Metasploit Framework

      4. Run cross–site scripting vulnerability tests

      5. Execute dictionary–based brute force attacks

      6. Exports results in plain text, CSV or HTML files

    7. Kismet

      8. Kismet is one of the most used ethical hacking tools. It works network detector, packet sniffer, and intrusion detection system for 802.11 wireless LANs. Kismet will work with any wireless card which supports raw monitoring mode and can sniff 802.11a, 802.11b, 802.11g, and 802.11n traffic. The program runs under Linux, FreeBSD, NetBSD, OpenBSD, and Mac OS X. The client can also run on Microsoft Windows.


    8. SQLMap

      9. SQLMap is another SQL vulnerability scanner bundled with Kali Linux distribution. This ethical hacking tool is dedicated to target and exploit web apps that use MS SQL Server as the backend database server. Using SQLMap you can:

      1. Test database schema

      2. Fingerprint remote database

      3. Brute force attack with a word list

      4. Direct shell & reverse shell


       SQLMap is available in multiple Unix distros where the Perl interpreter is installed, including :

      1. Linux

      2. Mac OS X & iOS

      3. FreeBSD

      6. Server–side request forgery
