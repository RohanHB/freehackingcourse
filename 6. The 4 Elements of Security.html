The 4 Elements of Security


    All attacks are an attempt to breach computer system security. Security consists of four basic elements,
    They are :

      1. Confidentiality

      2. Authenticity

      3. Integrity

      4. Availability


    1. Confidentiality :

      2. Stealing passwords or other data is a Confidentiality attack, because it allows someone other than the intended recipient to gain access.

    2. Authenticity :

      3. MAC address spoofing is an Authenticity attack because it allows an unauthorized device to connect to a network such as on a wireless one.

    3. Integrity :

      4. Bit–flipping attacks are considered Integrity attacks because the data may have been tampered with in transit or within the original system.

    4. Availability :

      5. With Denial–of–Service (DoS) attack, a hacker attacks the Availability elements of systems and networks.
