Operating System


    An Operating System ( OS ) is the software that directly manages a system's hardware and resources, like CPU, memory and storage. The OS sits between applications and hardware and makes the connections between all of your software and the physical resources that do the work. Think about an OS like a car engine. An engine can run on its own, but it becomes a functional car when it's connected with a transmission, axies and wheels. Without the engine running properly, the rest of the car won't work.


    Following are Some of the Important Functions of an OS

    1. Processor Management

    2. Memory Management

    3. Device Management

    4. File Management

    5. Security

    6. Control Over System Performance

    7. Error Detecting Aids

    8. Job Accounting

    9. Co–ordination between Other Software Users.
