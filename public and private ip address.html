Public and Private IP Adderss

    • There are two kinds of IP address : Public and Private.

    • A Public IP address is the address that identifies your device on the Internet.

    • Everyone else on the Internet can see your Public IP address.

    • A Private IP address is the address that identifies your device on your own personal network, like at home.

    • You can find your public address by googling my IP address and you can find your private address by opening a command prompt/terminal window and typing in ipconfig ( ifconfig for Mac ).


    Public & Private IP Addresses Cont. :

    • Private IP addresses fall into one of the following three ranges :

      1. 192.168.0.0 – 192.168.255.255 ( 65,536 IP addresses )

      2. 172.16.0.0 – 172.31.255.255 ( 1,048,576 IP addresses )

      3. 10.0.0.0 – 10.255.255.255 ( 16,777,216 IP addresses )

    • While you do not need to memorize these ranges, just be aware of what a public address looks like vs. a private address.
